# CampusBridge

A secure college mentorship and alumni networking platform that connects juniors, seniors, and alumni for career guidance, placement preparation, mentorship, study resources, and professional networking.

---

# 🚀 Overview

CampusBridge is a private college community platform designed to bridge the communication gap between:

* Juniors
* Seniors
* Passed-out Alumni

The platform helps students:

* Get career guidance
* Learn placement preparation strategies
* Access study resources
* Connect with mentors
* Explore internship opportunities
* Build professional networks safely

Unlike public social media platforms, CampusBridge focuses on:

* Verified college users
* Privacy protection
* Anti-spam systems
* Safe mentorship communication

---

# 🎯 Problem Statement

In many colleges:

* Juniors do not know seniors personally
* Alumni connections disappear after graduation
* Students lack proper career guidance
* Placement preparation knowledge is scattered
* Valuable experiences are not documented

CampusBridge solves this problem by creating a secure and structured mentorship ecosystem inside the college community.

# 🛡️ Anti-Misuse Features

To ensure student safety and prevent misuse:

* No public phone numbers
* No public personal emails
* Internal messaging system only
* Connection request approval system
* Spam detection
* Abuse keyword filtering
* Report & block system
* Daily message/request limits
* Reputation/trust score system

# 🔄 Application Flow

1. User registers using college email
2. Admin verifies alumni accounts
3. Users create profiles
4. Students ask doubts or request mentorship
5. Seniors/alumni respond
6. Users interact through secure messaging
7. Admin monitors activity and prevents misuse
